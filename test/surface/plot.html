Event Driven T-S Fuzzy Surface Plot

This tool help for tesing Node-RED T-S FIS. After set the the parameters (IDs, start and end values of test range) press Run Test button. The test values willl be sent via a WebSocket to the Node-RED FIS an the result surface plot will shown after all the output values received. If Node-RED is not running on localhost please modify the URL in app.js.
